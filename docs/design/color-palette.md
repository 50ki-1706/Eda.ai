# Color Palette & Theming (デザイン構造ガイドライン)

## 1. 共通セマンティック（The Trilogy: 状態の言語）

モードを問わず、色の持つ「意味」は共通です。

- **Primary / Stable**: `#00B4FF` (正常、完了)
- **Warning / Hesitation**: `#FFD700`
- **Error / Critical**: `#FF4B4B`

## 2. ライトモード

| 要素                | カラーコード | 役割・イメージ                                         |
| ------------------- | ------------ | ------------------------------------------------------ |
| **Main Background** | `#FFFFFF`    | 極めて純粋な白。                                       |
| **Sub Background**  | `#F5F7FA`    | サイドバーや入力フィールド。微かなグレーで階層を作る。 |
| **Border / Line**   | `#E0E0E0`    | ガラスのような薄い境界線。                             |
| **Main Text**       | `#1A1A1A`    | 鋭く、可読性の高い黒。                                 |
| **Accent UI**       | `#00B4FF`    | 青いLED。決定ボタンやアクティブ状態。                  |

## 3. ダークモード

| 要素                | カラーコード     | 役割・イメージ                                    |
| ------------------- | ---------------- | ------------------------------------------------- |
| **Main Background** | `#0A0A0A`        | 深い闇。システムに集中させるための Deep Black。   |
| **Sub Background**  | `#1A1A1A`        | カードやモーダル。背景より一段明るい黒。          |
| **Border / Line**   | `#4B4B4B`        |                                                   |
| **Main Text**       | `#E0E0E0`        | 目に優しいライトグレー。情報の階層を整理。        |
| **Accent UI**       | `#00B4FF` (Glow) | 暗闇で発光するLED。ホログラムのような質感を付与。 |
